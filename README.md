<h1 align="center">🤚 LinkedIn Gesture Launcher</h1>
<h3 align="center">Computer Vision Based Gesture Control</h3>

<hr/>

<h2>🚀 Project Overview</h2>
<p>
This project implements a gesture-controlled system that launches LinkedIn 
using real-time computer vision techniques.
</p>

<p>
The system detects specific hand gestures via webcam 
and triggers an automated browser action.
</p>

<hr/>

<h2>🧠 Core Technologies</h2>

<ul>
<li>OpenCV for video capture</li>
<li>Hand tracking / motion detection</li>
<li>Gesture recognition logic</li>
<li>Automation to open LinkedIn</li>
</ul>

<hr/>

<h2>📊 Workflow</h2>

<ol>
<li>Capture webcam stream</li>
<li>Detect and track hand movement</li>
<li>Identify predefined gesture pattern</li>
<li>Trigger LinkedIn launch automatically</li>
</ol>

<h2>⚙️ How to Run</h2>

<pre>

</pre>

Make the predefined gesture in front of the camera to trigger LinkedIn.
